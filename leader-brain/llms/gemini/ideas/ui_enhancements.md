# Future Ideas & Improvements - 2026-03-25

## 1. Global Glass-Panel Theme Utility
- **Idea:** Create a centralized utility or custom Tailwind plugin for `glass-panel` that automatically handles transparency and borders for both Light and Dark modes.
- **Goal:** Reduce repetitive code and ensure a consistent "glass" look across the entire application.

## 2. Interactive Chat Loading States
- **Idea:** Enhance the `Loader2` animation with a more interactive and visually interesting component, such as a pulsing sphere or AI-inspired waveform.
- **Goal:** Improve the perceived performance and engagement while waiting for AI responses.

## 3. Persistent Chat Session
- **Idea:** Save the chat history to local storage or a session cookie to maintain conversation context across page reloads.
- **Goal:** Enhance the user experience for recurring visitors and ongoing queries.

## 4. Mobile Chat Sidebar
- **Idea:** Optimize the sidebar layout for smaller screens, perhaps as a bottom-sheet style component on mobile for better accessibility.
- **Goal:** Improve the usability of the Chatbot for mobile users.
