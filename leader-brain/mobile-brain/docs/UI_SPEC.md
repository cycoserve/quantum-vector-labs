# UI Design Specification — QVL Mobile

## Visual Direction
- **Theme**: Dark Mode by default (mirroring the web's primary state).
- **Primary Color**: #20D3EE (Cyan-Blue)
- **Background**: #010918 (Space Blue)
- **Secondary Background**: #020D1F
- **Accents**: Purple-500 (glows)

## Themed Components (Simulated Glass)
- **Glass Card**: `bg-white/5 border border-white/10 rounded-2xl`
- **Primary Button**: `bg-[#20D3EE] rounded-xl py-4 flex items-center justify-center` (text: black, bold)
- **Inputs**: `bg-[#00000040] border border-[#20D3EE33] rounded-xl px-4 py-3 text-white`

## Screen Breakdown
1. **Splash Screen**: 
   - Logo: QVL Icon (Cyan)
   - Background: Deep Space Blue gradient
   - Title: "Quantum Vector Labs"
2. **Onboarding (3 Slides)**:
   - Slide 1: "Neural Horizon" - Manage GenAI models globally.
   - Slide 2: "Context-Aware" - AI memory via Leader Brain.
   - Slide 3: "Planetary Scale" - 24/7 autonomous operations.
3. **Login Page**:
   - Fields: Email, Password
   - OAuth: Sign in with Google / GitHub
   - Link: Sign up here
4. **Home (Dashboard)**:
   - Header: Profile Icon, QVL Logo
   - Main: Welcome message, API usage chart (placeholder), Recent AI activities.
   - Tabs: Dashboard, Account, Settings
